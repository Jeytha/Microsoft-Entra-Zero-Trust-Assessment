# ğŸ“˜ What Is Zero Trust?

**Zero Trust** is a modern security framework built on three core principles:

1. **Verify explicitly**  
   Always authenticate & authorize based on all available signals (identity, device, risk, location).

2. **Use least privilege access**  
   Give users the minimum rights requiredâ€”no broad or permanent permissions.

3. **Assume breach**  
   Architect systems as if an attacker is already inside; minimize blast radius.

Microsoft applies Zero Trust across:
- Identity & Access  
- Devices  
- Applications  
- Networks  
- Infrastructure  
- Data  
- Monitoring

# ğŸ› ï¸ Technical Background

The assessment tool uses:

- **Microsoft Graph API**  
  Queries identity, authentication, CA, roles, user risk, and sign-in data.

- **Azure Resource Manager (ARM)**  
  Reads configuration, device management, and Intune policy data.

- **ZeroTrustAssessment PowerShell Module**  
  A Microsoft-built module that:
  - Authenticates to Microsoft Graph  
  - Connects to Azure  
  - Runs ~200 security checks  
  - Produces a full HTML report  

The tool is **read-only** and **makes no configuration changes**.
